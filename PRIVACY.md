> 本隐私政策依据《PreConnect 软件许可协议》第 3 条"同意使用数据"之约定制定，是许可证的组成部分。阁下下载、安装或使用 PreConnect 软件，即表示阁下已阅读并同意本政策的全部条款。

PreConnect 是一款运行于 Windows 设备上的局域网硬件监控数据提供者软件，其设计用途是在阁下授权的前提下，向同一局域网内经阁下配对的设备提供本机硬件传感器数据。**本软件不向任何互联网服务器上报任何数据；所有数据传输均限于阁下的本地局域网络，或存储于阁下本机的文件系统中。**

## 2. 各类信息详细说明

### 2.1 设备标识符（DeviceId）

- **内容**：一个随机生成的 UUID 字符串（如 `c9dcbec2-7d77-4f6f-a5d2-b5e2d23f33ce`），软件首次运行时生成，此后保持不变。
- **存储路径**：`%LocalAppData%\PreConnect\connectivity-settings.json`
- **传输**：包含在每一条 UDP 广播包中，向当前局域网所有设备定期广播；也会随硬件数据接口响应体返回给已配对客户端。
- **用途**：用于唯一标识本机，不与任何个人身份信息直接关联。
- **注意**：该 ID 持久化且稳定，局域网内任意设备均可接收到此 ID，并据此在不同时间识别同一台主机的在线状态。

### 2.2 主机名（DeviceName）

- **内容**：Windows 计算机名，用作本机在局域网中的显示名称。
- **存储路径**：`%LocalAppData%\PreConnect\connectivity-settings.json`
- **传输**：UDP 广播包（局域网全播，无需认证）；HTTP 状态接口（无需认证）；配对响应。
- **注意**：计算机名可能包含阁下的姓名或个人标识。局域网内任何设备无需认证即可获取。阁下可在 Windows 系统设置中更改计算机名以减少相关信息的暴露。

### 2.3 本机局域网 IP 地址

- **内容**：当前活跃网络接口的 IPv4 地址。
- **传输**：UDP 广播包（局域网全播，无需认证）；HTTP 状态接口的响应内容。
- **注意**：IP 地址仅在阁下的本地局域网范围内可见，不会传输至互联网。在某些企业或隔离网络场景下，获取 IP 地址可能暴露网络拓扑信息。

### 2.4 操作系统版本

- **内容**：操作系统版本字符串（如 `Microsoft Windows NT 10.0.22621.0`）。
- **传输**：HTTP 状态接口（**无需认证**，局域网内任何能访问本软件服务端口的设备均可获取）。
- **注意**：局域网内任何能访问本软件服务端口的设备无需完成配对即可获取此信息。

### 2.5 硬件传感器数据（CPU / GPU / 内存 / 主板 / 存储）

- **内容（CPU）**：CPU 型号名称、制造商、核心数、各核心实时时钟频率（MHz）、核心温度（℃）、封装温度、功耗（W）、负载率（%）。
- **内容（GPU）**：GPU 型号名称、制造商（NVIDIA / AMD / Intel）、核心时钟、显存时钟、温度、显存使用量与总量。
- **内容（内存）**：已用内存（字节）、可用内存（字节）、总内存（字节）。
- **内容（主板）**：主板名称、各传感器温度。
- **内容（存储）**：硬盘名称、温度、SSD 剩余寿命、已用空间、总容量。
- **传输**：硬件数据接口（**必须提供有效的配对会话令牌方可访问**，未经配对的设备无法获取此数据）。
- **注意**：硬件型号可用于设备指纹识别；实时资源使用情况可在一定程度上反映阁下当前的设备使用模式；存储容量及剩余寿命属于设备资产信息。

### 2.6 已配对设备（客户端）的名称与设备 ID

- **内容**：在配对握手过程中，由客户端设备主动提供的显示名称和设备 UUID。
- **存储路径**：`%LocalAppData%\PreConnect\sessions.json`（明文 JSON 格式）。
- **存储内容示例**：
  ```json
  [
    {
      "Token": "<base64 随机令牌>",
      "DeviceId": "<客户端 UUID>",
      "DeviceName": "<客户端设备名>",
      "ExpiresAtUtc": "2126-03-16T00:00:00+00:00"
    }
  ]
  ```
- **用途**：用于标识哪些设备已获得访问授权，以便在会话验证时核对身份。
- **注意**：会话令牌以明文存储，任何拥有阁下账户访问权限的程序均可读取该文件。当前版本令牌的有效期设定极长（近似于长期有效）。阁下可删除 `sessions.json` 文件来吊销全部已配对设备的访问权限。客户端设备名可能包含对端用户的姓名或个人信息。

### 2.7 会话令牌（Session Token）

- **内容**：256 位随机认证凭据（Base64 编码），配对成功后颁发。
- **存储路径**：`%LocalAppData%\PreConnect\sessions.json`（明文 JSON 格式）。
- **传输**：配对完成后随响应返回给客户端；客户端后续访问硬件数据接口时须随请求一并提供（通过 HTTP Header 或 URL 查询参数）。
- **注意**：通过 URL 查询参数传递令牌时，令牌可能被网络代理日志或浏览器历史记录，建议优先使用 HTTP Header 方式传递。

### 2.8 启动诊断日志

- **内容**：每次启动及发生未处理异常时，软件会在本机写入诊断日志，包含以下信息：
  - UTC 时间戳
  - 软件可执行文件的完整路径（包含 Windows 用户名片段）
  - .NET 运行时版本、进程架构、操作系统版本
  - 是否以管理员身份运行
  - 异常完整堆栈跟踪信息（可能包含本地文件绝对路径）
- **存储路径**：`%LocalAppData%\PreConnect\Logs\startup-YYYYMMDD.log`（按日期分文件）。
- **传输**：仅存储于本机，不通过任何方式传输至网络。
- **注意**：诊断日志中可能含有文件路径（其中包含阁下的 Windows 用户名）。日志文件不设大小上限，阁下可自行前往上述路径手动清理。

### 2.9 开机自启注册表项

- **内容**：软件可执行文件的完整路径，写入 Windows 注册表自启动项（`HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run\PreConnect`）。
- **触发条件**：仅在阁下主动启用"开机自启"功能时写入，软件安装时不自动配置。
- **注意**：可执行文件路径中可能包含阁下的 Windows 用户名。禁用开机自启后，本软件将自动删除该注册表项。

## 3. 数据传输范围说明

```
局域网内任何设备
        │
        │  UDP 广播（255.255.255.255 / 224.0.0.251，端口 53530）
        ▼
【无需认证即可接收】
    • deviceId（设备标识符）
    • name（主机名/计算机名）
    • IP 地址
    • 服务端口号

        │  HTTP 请求（默认端口 5005）
        ▼
/api/ping     ← 无需认证  →  程序名称 + 当前时间
/api/status   ← 无需认证  →  主机名、OS 版本、IP 地址、端口、活跃连接数
/api/pair     ← 需提供 PIN →  接收客户端设备名/ID；返回会话令牌
/api/telemetry← 需会话令牌 →  返回完整硬件传感器快照

本机存储（%LocalAppData%\PreConnect\）
    connectivity-settings.json  ←→  设备标识符、主机名、端口配置
    sessions.json               ←→  会话令牌（明文）及已配对设备信息
    Logs\startup-YYYYMMDD.log   ←→  诊断日志（含路径及异常栈）
```

## 4. 当前版本已知事项

阁下使用前应知悉以下事项：

| 事项 | 说明 |
|---|---|
| 无需认证的接口 | HTTP 状态接口在无任何认证的情况下，向局域网内所有设备暴露主机名和操作系统版本信息。 |
| 会话令牌明文存储 | 会话令牌以明文 JSON 格式存储于本机，未经加密处理。 |
| 会话令牌长期有效 | 当前版本颁发的会话令牌有效期设置极长，近似于长期有效。如需吊销，请删除 `sessions.json` 文件。 |
| 令牌 URL 参数传递 | 软件支持通过 URL 查询参数传递会话令牌，此方式下令牌可能被网络代理日志或浏览器记录。建议使用 HTTP Header 方式传递。 |
| 发现广播频率 | 发现广播每 2 秒发出一次，接收方无法被过滤。 |
| 诊断日志无自动清理 | 诊断日志文件不设大小或数量上限，长期运行后可能积累，阁下可自行手动清理。 |

## 5. 软件不收集的信息

以下信息**明确不被本软件收集或传输**：

- 用户账户密码、Windows 登录凭据
- 浏览器历史、剪贴板内容
- 文件系统目录结构或文件内容
- 键盘输入、鼠标行为
- 麦克风、摄像头、位置信息
- 任何形式的云端上报或遥测推送
- 进程列表或正在运行的应用程序

## 6. 本地数据存储汇总

| 存储位置 | 内容摘要 | 是否加密 | 清除方式 |
|---|---|---|---|
| `%LocalAppData%\PreConnect\connectivity-settings.json` | 设备标识符、主机名、端口配置 | 否（明文 JSON） | 手动删除文件 |

| `%LocalAppData%\PreConnect\Logs\startup-*.log` | 启动及异常诊断日志 | 否（明文文本） | 手动删除文件 |
| `HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run\PreConnect` | 软件可执行文件完整路径（开机自启条目） | 否 | 在软件中禁用开机自启即自动删除 |

## 7. 阁下的权利

依据中华人民共和国《个人信息保护法》及其他适用法律，阁下就本软件在本机处理的与阁下相关的信息享有以下权利：

- **查阅权**：相关文件路径已在第 6 条列明，阁下可直接访问并查阅本软件在阁下设备上存储的数据。
- **更正权**：阁下可直接编辑 `connectivity-settings.json` 中的设备名称等配置信息，或通过软件界面进行修改。
- **删除权**：阁下可随时停止使用并手动删除上述所有本地数据文件，以清除本软件在阁下设备上保留的全部信息。
- **撤回同意**：阁下可随时通过软件界面关闭局域网发现广播功能，或卸载软件并删除上述数据文件，以撤回对本政策的全部同意。

由于本软件不将任何数据传输至互联网或第三方服务器，上述权利均可由阁下在本机直接行使，无需向作者提出请求。如有任何疑问，阁下亦可通过第 11 条所列方式联系作者。
